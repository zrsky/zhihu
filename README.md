# 知乎
## 技术栈
* HTML5+CSS3+JS+jquery
* node.js(express)
* mongodb（mongoose）

## 功能
1. 基本框架，node连接mongodb，web正常返回html；
2. 注册，登录页面：canvas实现验证码；注册，登录功能正常；session记录状态；
3. 问题页面：新建，编辑，删除，赞同，反对，感谢答案；
4. 个人页面：显示提问，回答，动态；
5. 主页面：显示关注人的动态；
